# MedSync PWA Icons

This directory contains placeholder icons for the MedSync PWA (Progressive Web App). 

## Icons Needed

Replace the following placeholder files with actual PNG icons:

- **icon-192x192.png** - Standard app icon (192x192px)
- **icon-512x512.png** - Large app icon for splash screens (512x512px)
- **icon-192x192-maskable.png** - Maskable icon for adaptive display (192x192px)
- **icon-512x512-maskable.png** - Large maskable icon (512x512px)
- **search-96x96.png** - App shortcut icon for Patient Search (96x96px)
- **vitals-96x96.png** - App shortcut icon for Vitals Entry (96x96px)

## Icon Requirements

### Standard Icons
- **Format:** PNG with transparent background
- **Sizes:** 192x192px and 512x512px
- **Purpose:** General app icon

### Maskable Icons
- **Format:** PNG with transparent background
- **Sizes:** 192x192px and 512x512px
- **Purpose:** Adaptive display on devices that support maskable icons
- **Design:** Icons should be square and fill the entire canvas, as the device will apply the mask

### Shortcut Icons
- **Format:** PNG with transparent background
- **Size:** 96x96px
- **Purpose:** Quick action shortcuts in app drawer
